This repository contains a curated collection of problems designed to help learners strengthen their understanding of Pandas, the powerful Python library for data manipulation and analysis. Each problem focuses on real world scenarios so learners can build confidence in working with DataFrames, data cleaning, aggregation, indexing, and more.

## Purpose

The goal of this repository is to provide a clear path for practicing Pandas. Whether you are preparing for interviews, improving your data analysis skills, or revising core concepts, these problems will guide you step by step and help you master the fundamentals.

## What You Will Learn

* Working with Series and DataFrames
* Importing and exploring datasets
* Data cleaning and preprocessing
* Filtering, sorting, and indexing
* Groupby operations and aggregation
* Merging, joining, and concatenation
* Handling missing data
* Applying functions and lambda expressions
* Realistic analytical problem solving

## Repository Structure

* **problems/**
  Contains all Pandas exercises organized with clear filenames.

* **solutions/**
  Includes solution files for each problem for reference and learning.

* **data/**
  Contains sample datasets used throughout the exercises.

## How to Use This Repository

1. Choose any problem from the problems folder.
2. Attempt to solve it on your own.
3. Compare your solution with the official one in the solutions folder.
4. Practice regularly to strengthen your data handling skills.

## Requirements

Make sure you have the following installed:

```bash
python 3.x
pandas
```

Install Pandas using:

```bash
pip install pandas
```
